# Sistema de Parqueadero - Autos Colombia

Este sistema permite registrar la entrada y salida de vehículos en el parqueadero por mensualidad.

## Tecnologías
- Node.js + Express
- PostgreSQL
- REST API

## Endpoints
- POST /api/registros/entrada
- POST /api/registros/salida

## Instalación
1. Clonar repositorio
2. Configurar `.env` con credenciales de PostgreSQL
3. Ejecutar `npm install`
4. Correr con `node app.js`
